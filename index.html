IIViewer

专为图像信号处理开发者设计的实用工具

English

关于

这个仓库专为打开和查看 ISP 中间图像而设计。我们支持多种图像格式，按类别分类如下：

常见图像格式

jpg
bmp
png
tiff

灰度/彩色图像

pgm (8/10/12/14/16 bit)
pnm (8/10/12/14/16 bit)

RAW 格式

raw (8/10/12/14/16/18/20/22/24 bit)
mipi-raw (10/12/14 bit)
rgbir-raw (8/10/12/14/16/18/20/22/24 bit)

YUV 格式

yuv (8/10/12 444-interleave4 444-plannar 422-UYVY 422-YUYV 420-NV12 420-NV21 420P-YU12 420P-YV12 400)

HEIF 格式

heif (.heic 8bit yuv420/422/444)

跨平台支持

IIViewer 支持多种操作系统，提供原生体验：

Windows

原生 Windows 应用，支持 x64 架构

Ubuntu

DEB 包安装，完美适配 Linux 环境

macOS

原生 macOS 应用，支持最新版本

主界面

Ubuntu 中文界面

macOS 中文界面

按照提示，将任何支持格式的图像拖到虚线矩形中，它将显示图像内容。当您通过滚动鼠标滚轮放大到 96X 时，您将看到每个像素的真实值。

从源代码构建

Windows (AMD64)

准备

  1. 安装 cmake (>=3.20)
  2. 安装 MSVC 或 MinGW64（应支持 C++17 至少）
  3. 构建并安装 OpenSSL，从官方 仓库 和 构建指南，成功后将其安装目录复制到此 git 仓库的 thirdparty 目录。
  4. 构建 libde265（请参考 README.md 中的详细步骤）
  5. 构建 libheif（请参考 README.md 中的详细步骤）

有关详细构建步骤，请查看 完整 README.md。
